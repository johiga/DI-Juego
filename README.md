# DI-Juego
Proyecto para desarrollo de interfaces que consiste en un juego tradicional pasado al ordenador.

Este juego es el MasterMind. Que intenta descifrar una serie de diferentes colores.
Tendremos pistas que nos indiquen si los colores son correctos y también si la posición es correcta.
El color blanco indica que el color está pero no en la posición correcta.
El color negro indica que tanto el color como la posición son correctas.
